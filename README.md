# ADL-CW2

## 环境配置

使用 Conda 创建并配置环境：
```bash
# 创建环境
conda create -n comp0197-cw2-pt python=3.12 pip

# 激活环境
conda activate comp0197-cw2-pt 

# 安装 PyTorch
pip install torch==2.5.0 torchvision --index-url https://download.pytorch.org/whl/cpu
```
